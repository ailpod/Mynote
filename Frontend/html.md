

## 基础与文本格式化

在 VS Code 中输入 `!(eg: em h1)` 并按 `Tab` 键，可以快速生成以下所有的骨架代码。

------

### 1. 文档基础结构标签

在写具体内容之前，这些是每个 HTML 文件必须具备的“骨架”。

| **标签**     | **原意**                  | **视觉效果**     | **语义含义及用法**                                   |
| ------------ | ------------------------- | ---------------- | ---------------------------------------------------- |
| `<!DOCTYPE>` | Document Type             | 无               | 声明文档类型，告诉浏览器这是 HTML5 标准。            |
| `<html>`     | HyperText Markup Language | 无               | **根标签**，包含整个页面的所有内容。                 |
| `<head>`     | Head                      | 无               | **头部**，存放元数据（如字符编码、标题、CSS 链接）。 |
| `<meta>`     | Metadata                  | 无               | **元信息**，常用 `<meta charset="UTF-8">` 设置编码。 |
| `<title>`    | Title                     | 浏览器标签栏文字 | **标题**，显示在浏览器选项卡上，对 SEO 极重要。      |
| `<body>`     | Body                      | 页面主体区域     | **主体**，包含所有用户可见的内容。                   |

**eg :**

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    页面可见内容
</body>
</html>
```

------

### 2. 基础排版标签

这些标签用于构建页面的基本轮廓（块级结构）。

| **标签**        | **原意**        | **视觉效果**             | **语义含义及用法**                           |
| --------------- | --------------- | ------------------------ | -------------------------------------------- |
| `<h1>` - `<h6>` | Heading 1 - 6   | 字体加粗、变大 (h1 最大) | **标题级别**，用于区分内容的层级结构。       |
| `<p>`           | Paragraph       | 段落间有间距             | **段落**，最常用的文本组织方式。             |
| `<br>`          | Break           | 强制换行                 | **换行**，单标签，用于在段落内强行换一行。   |
| `<hr>`          | Horizontal Rule | 水平分隔线               | **水平线**，单标签，用于区分不同的主题板块。 |

**eg :**

```html
<h1>一级标题</h1>
<p>第一行文字<br>第二行文字</p>
<hr>
```

### 3. 强调与文本格式化标签

这些用于区分“样式”与“含义”。

| **标签**   | **原意**           | **视觉效果**  | **语义含义及用法**                             |
| ---------- | ------------------ | ------------- | ---------------------------------------------- |
| `<strong>` | Strong Importance  | **加粗**      | **重要性**，表示内容非常关键、紧急或严肃。     |
| `<b>`      | Bold               | **加粗**      | **引起注意**，纯视觉上的加粗，没有特殊含义。   |
| `<em>`     | Emphasis           | *斜体*        | **强调**，表示重读，类似于说话时的重音语气。   |
| `<i>`      | Idiomatic / Italic | *斜体*        | **文体区别**，用于术语、外语、内心活动等。     |
| `<del>`    | Delete             | ~~删除线~~    | **被删除内容**，常用于显示旧价格或修过的内容。 |
| `<ins>`    | Insert             | <u>下划线</u> | **插入内容**，通常与 `<del>` 配合使用。        |

**eg :**

```html
<strong>重要文字</strong>
<em>重读文字</em>
<del>原价 99</del> <ins>现价 19</ins>
```

### 4. 块级元素与行内元素

在 HTML 中，元素默认分为两大类：**块级元素 (Block)** 和 **内联/行内元素 (Inline)**。它们决定了标签在页面上是如何“排队”的。

------

#### 4.1. 块级元素 (Block Elements)

**特点**：独占一行；宽度默认撑满父容器；可以设置宽高；可以包含其他块级或行内元素。

| **标签**      | **原意**       | **语义含义及用法**                                           |
| ------------- | -------------- | ------------------------------------------------------------ |
| `<div>`       | Division       | **分区/容器**：最通用的块级容器，用于组合其他元素，无特殊语义。 |
| `<h1>`-`<h6>` | Heading        | **标题**：默认块级，独占一行。                               |
| `<p>`         | Paragraph      | **段落**：文本的基础块状单元。                               |
| `<ul>`        | Unordered List | **无序列表**：整个列表是一个块。                             |
| `<ol>`        | Ordered List   | **有序列表**：整个列表是一个块。                             |
| `<li>`        | List Item      | **列表项**：每个列表项也是独占一行的块。                     |
| `<header>`    | Header         | **页头**：HTML5 语义化标签，表示页面或区域的头部。           |
| `<footer>`    | Footer         | **页脚**：表示页面或区域的底部信息。                         |

**eg :**

```html
<div>
    <h1>文章标题</h1>
    <p>文章内容</p>
</div>
```

------

#### 4.2. 内联/行内元素 (Inline Elements)

**特点**：不会另起一行，只占据内容所需的空间；**无法直接设置宽高**；通常只包含文本或其他行内元素。

| **标签**   | **原意** | **语义含义及用法**                                           |
| ---------- | -------- | ------------------------------------------------------------ |
| `<span>`   | Span     | **范围/跨度**：最通用的行内容器，常用于包裹一小段文字以便单独设置样式。 |
| `<a>`      | Anchor   | **锚点/超链接**：点击跳转，默认行内显示。                    |
| `<strong>` | Strong   | **加粗/重要性**：强调文本内容。                              |
| `<em>`     | Emphasis | **斜体/强调**：重读文本内容。                                |
| `<img>`    | Image    | **图像**：虽然是行内元素，但可以设置宽高（特殊属性）。       |
| `<code>`   | Code     | **代码**：展示一小段计算机代码。                             |

**eg :**

```html
<span>行内文字</span>
<a href="#">超链接</a>
```

------

#### 4.3. 核心区别对比表

| **特性**     | **块级元素 (Block)**               | **行内元素 (Inline)**              |
| ------------ | ---------------------------------- | ---------------------------------- |
| **换行**     | **自动换行**（霸道，必须独占一行） | **不换行**（合群，跟别人排成一排） |
| **宽高设置** | **有效**（可手动调节长宽）         | **无效**（宽度由内容多少决定）     |
| **嵌套关系** | 可以容纳块级和行内元素             | **通常只能容纳文本或行内元素**     |
| **代表标签** | `div`, `p`, `h1-h6`, `li`          | `span`, `a`, `strong`, `i`         |

### 5. 图像标签与常用格式

#### 5.1. 图像标签基础

> **快速生成**：在 VS Code 中输入 `img` 然后按 `Tab` 键，会自动生成 `src` 和 `alt` 属性。

| **标签/属性** | **原意**       | **视觉效果**             | **语义含义及用法**                                           |
| ------------- | -------------- | ------------------------ | ------------------------------------------------------------ |
| `<img>`       | Image          | 显示一张图片             | **图像标签**：单标签，用于在页面中嵌入图片。                 |
| `src`         | Source         | 路径对应的图像           | **必须属性**：指定图片的存储路径（网络地址或本地路径）。     |
| `alt`         | Alternate Text | 图片加载失败时显示的文字 | **替换文本**：搜索引擎靠它读懂图片内容，也是盲人阅读器的核心参考。 |
| `title`       | Title          | 鼠标悬停时显示的文字     | **提示文本**：给用户的额外信息提示。                         |
| `width`       | Width          | 改变图片宽度             | **宽度**：单位通常是像素 (px)，建议只设置宽或高中一个，让图片等比缩放。 |
| `height`      | Height         | 改变图片高度             | **高度**：若同时设置宽高且比例不对，图片会变形。             |

**eg :**

```html
<img src="./example.jpg" alt="示例图片" title="鼠标悬停提示">
```

------

#### 5.2. 常见图像格式对比

在前端开发中，选择正确的格式能显著提升网页加载速度。

| **格式**       | **全称**                         | **优点**                                                 | **缺点**                                 | **适用场景**                         |
| -------------- | -------------------------------- | -------------------------------------------------------- | ---------------------------------------- | ------------------------------------ |
| **JPG / JPEG** | Joint Photographic Experts Group | 色彩丰富，压缩率高，文件小。                             | 不支持透明，多次保存会有损。             | **摄影照片**、复杂的彩色大图。       |
| **PNG**        | Portable Network Graphics        | **支持透明**，无损压缩，边缘清晰。                       | 文件体积通常比 JPG 大。                  | **Logo**、图标、需要透明背景的素材。 |
| **GIF**        | Graphics Interchange Format      | **支持动画**，支持简单透明。                             | 仅支持 256 色，画质一般。                | 简单的表情包、小动画效果。           |
| **WebP**       | Web Picture                      | **现代全能型**：体积比 JPG 小 30% 以上，支持透明和动画。 | 老版本浏览器兼容性稍差（目前基本普及）。 | **网页首选**，兼顾画质与体积。       |
| **SVG**        | Scalable Vector Graphics         | **矢量图**：无限放大不模糊，文件极小。                   | 不适合表现复杂色彩的照片。               | **图标 (Icons)**、简单的插画、Logo。 |

### 6. 音视频标签

#### 6.1. 音频标签 `<audio>`

| **标签/属性** | **原意** | **语义含义及用法**                                           |
| :------------ | -------- | :----------------------------------------------------------- |
| `<audio>`     | Audio    | **音频标签**：用于在页面中嵌入音频文件（如 MP3）。           |
| `src`         | Source   | **音频路径**：音频文件的存储位置（相对或绝对路径）。         |
| `controls`    | Controls | **控制面板**：显示播放、暂停、进度条、音量等控件。           |
| `autoplay`    | Autoplay | **自动播放**：页面加载完后立即播放（受浏览器静音策略限制）。 |
| `muted`       | Muted    | **静音**：默认关闭声音播放。                                 |
| `loop`        | Loop     | **循环播放**：音频播放结束后自动重新开始。                   |

**eg :**

```html
<audio src="./music.mp3" controls loop>您的浏览器不支持音频</audio>
```



------

#### 6.2. 视频标签 `<video>`

| **标签/属性**      | **原意**       | **语义含义及用法**                                 |
| ------------------ | -------------- | -------------------------------------------------- |
| `<video>`          | Video          | **视频标签**：用于在页面中嵌入视频文件（如 MP4）。 |
| `src`              | Source         | **视频路径**：视频文件的存储位置。                 |
| `controls`         | Controls       | **控制面板**：显示视频播放器的操作界面。           |
| `poster`           | Poster         | **封面/海报**：在视频播放前显示的预览图。          |
| `width` / `height` | Width / Height | **尺寸**：设置视频播放器的显示宽度和高度。         |
| `muted`            | Muted          | **静音**：想要自动播放视频通常必须静音。           |

**eg :**

```html
<video src="./movie.mp4" controls width="600" poster="./cover.jpg" muted autoplay loop>
    您的浏览器不支持视频播放。
</video>
```

### 7. 超链接标签 `<a>`

超链接属于**行内元素**，但它非常特殊，内部可以包裹文本、图片甚至整个块级区域（HTML5 支持）。

#### 7.1. 基础属性

| **属性** | **原意**            | **语义含义及用法**                                           |
| -------- | ------------------- | ------------------------------------------------------------ |
| `href`   | Hypertext Reference | **必须属性**：指定跳转的目标地址（URL、路径或锚点）。        |
| `target` | Target              | **打开方式**：规定在何处打开链接。常用 `_self`（原窗口）和 `_blank`（新窗口）。 |
| `title`  | Title               | **悬停提示**：鼠标指向链接时显示的辅助文字。                 |

**eg :**

```html
<a href="https://www.google.com" target="_blank">点击访问 Google</a>

<a href="about.html">关于我们</a>

<a href="#">待定链接</a>

<a href="./download.zip">下载链接</a>

<a href="mailto:example@qq.com">联系我们</a>
```

------

#### 7.2. 锚点链接 (Anchor Link)

用于实现**页面内部**的快速跳转（例如“回到顶部”或点击目录跳转到指定章节）。

**实现步骤：**

1. **设置目标**：给目标标签添加一个唯一的 `id` 属性。
2. **设置链接**：在 `<a>` 标签的 `href` 属性中使用 `#id名称`。

**eg :**

```html
<a href="#chapter3">跳到第三章</a>

<h2 id="chapter3">第三章：锚点详解</h2>
```

------

#### 7.3. 特殊链接功能

除了网页跳转，`<a>` 标签还支持触发一些系统级操作。

**eg :**

```html
<a href="tel:10086">拨打客服电话</a>

<a href="mailto:example@gmail.com">发邮件给我</a>

<a href="document.pdf" download>下载 PDF 报告</a>
```

------

### 8. 路径（链接通用）

在使用 `href` 或 `src` 时，路径的正确性至关重要：

| **路径类型** | **写法**                  | **含义**                       |
| ------------ | ------------------------- | ------------------------------ |
| **绝对路径** | `https://...` 或 `C:/...` | 完整的网络地址或本地盘符地址。 |
| **同级路径** | `index.html`              | 直接写文件名。                 |
| **下级路径** | `html/index.html`         | 文件夹名 / 文件名。            |
| **上级路径** | `../index.html`           | `../` 表示返回上一层文件夹。   |

------

小贴士：

- **禁止空跳转**：如果写了 `<a href=""></a>`，点击它通常会刷新当前页面。建议使用 `<a href="#"></a>` 来代替。
- **图片超链接**：可以把 `<img>` 嵌套在 `<a>` 里面，这样点击图片就能实现跳转。

**eg :**

```HTML
<a href="https://www.youtube.com">
    <img src="play-button.png" alt="播放视频">
</a>
```
