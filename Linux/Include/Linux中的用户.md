> - 删除用户组  ：groupdel  用户组名

## **3.  权限信息**
 
>[!tip]+ 在使用 ls 查看目录信息后     
>
> - 第一部分会显示例如：drwxr-xr-x的信息（记录为一个数组 T[9]） 
> 
> - 第二部分则是该目录所属用户
> 
> - 第三部分是该目录所属用户组
> 
>>[!example]+ 数组各部分含义 
>> -  T[0] 可显示： - (文件)   d (文件夹)   l (软链接)
>> 
>> -  T[1,2,3] 表示所属用户权限 ，可显示：r (可读)   x (可写)   w (可执行)   - (填充空位)
>> 
>>  - T[4,5,6] 表示所属用户组权限 ，可显示：r  x  w  - 
>>      
>> - T[7,8,9] 表示其他用户权限 ，可显示：r  x  w  -

## **4. 修改权限命令**

```
    chmod [ - R ]  权限  文件/文件夹 
```

-  选项 -R 对文件夹内全部内容进行同样操作

>[!example]-  示例
>
>chmod u(用户)=rwx , g(组)=rx , o(其他用户)=x  test.txt
> 
>==可使用数字更改权限，eg： chmod  751  test.txt（等于上面的示例）==
> 
>==其中751为二进制，7 = 111（rwx） ，5 = 101 （r-x），1 =001 （--x）==

- chown  - R    用户   ：  用户组   文件/文件夹    ：修改文件、文件夹所属用户、用户组
