**命令的通用格式**

```
 command  [-options]  [parameter]       注：语法中的[ ]表示可选的意思
```

- command：命令本身

- options：命令的一些选项 ，用于控制命令的行为细节 [可选项]	

- parameter：命令的参数，一般用于命令的指向目标 [可选项]

**eg：

```
· ls (命令本身) -l (选项)  /home/AaTool (参数) 
```

- 表示以列表形式显示/home/AaTool 中的内容

简单命令的使用

## **1.  ls**

**作用是列出目录下的内容，使用 ls / 可以列出根目录的内容**

```
ls [-a -l -h]  [path] 
```

- 无参数时，以平铺形式展示目录内容

>[!example]- 可选项
> - -a（all）选项可以显示目录下所有文件（包括隐藏文件夹，其名称前有“ . ")
> 
> - -l 选项表示以列表竖向形式展示内容，并显示更多信息
> 
> - -h选项可以用于展示当前文件大小（默认是bit，使用 -h后可以显示 K ， M ，G）

- -选项可以组合使用 ，==eg：-l-a    -la    -al   -lh

---

## **2. cd**

**更改当前的工作目录** (change directory)^[[[操作符|特殊路径符]]]

```
cd [Linux路径]
```

- cd命令无参数时 ， 切换到根目录

- cd .. (返回上级目录)		cd ../..  (返回前两级目录 )

- cd ./path (切换当前目录下的path)       cd ~ (切换到home目录 )

---

## **3. pwd**

**查看当前工作目录 （print work directory）**

```
pwd
```

---

## **4.  绝对/ 相对路径**

-  绝对路径 ：以根目录为起点描述路径，以 / 开头          

```
eg :  cd /home/ye/OsLab2
```

-  相对路径 ：以当前目录为起点描述路径                        

```
eg : cd OsLab2  (前提要在/home/ye 目录下，否则找不到OsLab2)
```

## **5. mkdir**

**创建一个文件夹（make directory）**

```
mkdir  [-p]  path
```

-   path 参数必填，相对/绝对路径都可以

-  -p选项表示自动创建不存在的父目录     

-  ==eg ：  mkdir -p  home/OsLab2/demo==

## **7. 文件操作命令二**

### **cp**

**复制文件/文件夹（copy）**

```
 cp  [ -r ]  path1  path2
```

-  -r 选项用于复制文件夹（无该参数时复制文件）

-  path1 表示被复制的文件或文件夹

-  path2 表示复制去的地方


### **mv**

**移动文件/文件夹（move）**

```
mv path1 path2
```

-  path1 表示被移动的文件或文件夹

-  path2 表示移动去的地方


### **rm**

**删除文件/文件夹（remove） **     

```
rm [ -r  -f ]    path1 path2 ... path n
```

-  -r 选项用于删除文件夹（ -f 表示强制删除，不提示确认消息）

-  path 表示被删除的文件或文件夹  ，用空格隔开

-  ==rm 支持[[操作符|通配符]]来做模糊匹配== ^[[[操作符|通配符]]]

## **8. which**

**查找命令的程序文件存放的位置**

```
which + 命令` 
```

==只有可执行文件会被查到，cd等shell命令无法查到==

---------------------------------------------------------------------------------

## **9. find**

**按文件名查找文件**

-  按文件名查找：
```
    find + path(起始)   -name  "被查找文件名"
```

- 按文件大小查找：
```
    find + path(起始)   -size  + |  +- n[kMG]  ( 大于或小于，数字[大小单位] )
```

---

## **10. gerp**

**从文件中查找带关键字的行  (global regular expression print)**

```
grep [-n]  keyword  path
```

- -n可选项，可选显示包含keyword的行数

- 无path，可作输入输出端口

- ==可做[[操作符|管道符]]操作== ^[[[操作符|管道符]]]     
  

---

## **11. wc**

**统计文件的行数、字符数......  (word count)**

```
wc [-c -m -l -w]  path
```

>[!tip]- 可选项
>
> - -c   统计bytes数量
> 
> - -m  统计字符数量
> 
> - -l  统计行数
> 
> - -w  统计单词数量

- path不写的话，可作为内容输入端口（可用于管道符操作）

---

## **12. echo**

**在命令行内输出内容**

```
echo 输出内容
```

---

## **13. tail**

**可以查看文件的尾部内容，跟踪文件最新更改**

```
tail [ -n -f  ]  path
```

- -f   表示持续跟踪

- -n  表示查看尾部n行，无参时默认10行

- 组合使用-f和-num时候，应如此 ：tail  -n 3  -f  path
